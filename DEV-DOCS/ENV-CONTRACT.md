# Environment Contract

## Mobile Toolchain

Pinned in `luna_lighthouse/toolchain.env`:

- Flutter stable at framework revision `ea121f8859e4b13e47a8f845e4586164519588bc`
- Java 17
- Node 20
- Ruby 2.7.6
- Bundler 2
- Xcode 15 or newer on macOS

## Flutter Generated Environment

`luna_lighthouse/lib/system/environment.dart` is generated by `environment_config`. Regenerate it with:

```bash
cd luna_lighthouse
dart run environment_config:generate
```

## GitHub Release Secrets

Release workflows require Android signing secrets and Apple/Fastlane Match secrets documented in `docs/owner-requirements.md`.

## Notification Service Runtime

Required phase-2 variables include:

- Firebase: `FIREBASE_PROJECT_ID`, `FIREBASE_CLIENT_EMAIL`, `FIREBASE_DATABASE_URL`, `FIREBASE_PRIVATE_KEY`
- API keys: `FANART_TV_API_KEY`, `THEMOVIEDB_API_KEY`
- Redis: `REDIS_HOST`, `REDIS_PORT`, `REDIS_USE_TLS`, `REDIS_USER`, `REDIS_PASS`
- Service: `PORT`

These are not required for phase-1 mobile release unless reactivating hosted webhook paths.
