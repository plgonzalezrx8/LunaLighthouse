# Cloud Account

Cloud account features are **temporarily unavailable** during the phase-1 LunaLighthouse mobile relaunch.

{% hint style="info" %}
This deferment is intentional while backend and webhook infrastructure is rebranded and modernized under LunaLighthouse ownership.
{% endhint %}

## What Still Works

- Settings shows Cloud Account and Cloud Sync as disabled Coming Soon items so users can see that hosted cloud features are planned.
- Local on-device backup and restore.
- Core module connectivity for supported self-hosted services.

## Future Re-Enable

Cloud account functionality will return in a later phase after runtime upgrades, secret ownership transfer, and integration validation are complete.
