# <img width="40px" src="./assets/images/branding_logo.png" alt="LunaLighthouse"></img>&nbsp;&nbsp;LunaLighthouse

LunaLighthouse is an open source self-hosted controller focused on reliable mobile control of media services.

## Supported Modules

- [Lidarr](https://github.com/lidarr/lidarr)
- [Radarr](https://github.com/radarr/radarr)
- [Sonarr](https://github.com/sonarr/sonarr)
- [NZBGet](https://github.com/nzbget/nzbget)
- [SABnzbd](https://github.com/sabnzbd/sabnzbd)
- [Newznab Search](https://newznab.readthedocs.io/en/latest/misc/api/)
- [NZBHydra2](https://github.com/theotherp/nzbhydra2)
- [Tautulli](https://github.com/Tautulli/Tautulli)
- [Wake on LAN](https://en.wikipedia.org/wiki/Wake-on-LAN)

## Relaunch Scope

Phase-1 relaunch is mobile-first (iOS/Android).

- App identity: `LunaLighthouse`
- Mobile identifiers: `app.lunalighthouse.lunalighthouse`
- Domains: `*.lunalighthouse.app`
- Deferred features: cloud account and hosted webhook integrations
