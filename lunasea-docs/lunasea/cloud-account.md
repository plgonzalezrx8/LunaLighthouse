# Cloud Account

Cloud account features are **temporarily unavailable** during the phase-1 Lunarr mobile relaunch.

{% hint style="info" %}
This deferment is intentional while backend and webhook infrastructure is rebranded and modernized under Lunarr ownership.
{% endhint %}

## What Still Works

- Local on-device backup and restore.
- Core module connectivity for supported self-hosted services.

## Future Re-Enable

Cloud account functionality will return in a later phase after runtime upgrades, secret ownership transfer, and integration validation are complete.
