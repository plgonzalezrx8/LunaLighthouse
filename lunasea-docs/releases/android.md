# Android

Lunarr is available on Android 7.0+.

Note that Google Play Services are not required for core functionality. Hosted cloud account and webhook notification features are temporarily unavailable during phase-1 relaunch.

{% hint style="info" %}
If you want a stable experience, stick with stable releases. Want to test new builds of Lunarr? Read about the [build channels](../getting-started/build-channels.md) to make the right choice!
{% endhint %}

## Google Play Store

_Channel(s): `Stable`, `Beta`, `Edge`_

The easiest way for most users with Android devices would be to download releases of Lunarr directly from the [Google Play Store](https://www.lunarr.app/playstore)!

{% tabs %}
{% tab title="Stable" %}
Head to the [Google Play Store](https://www.lunarr.app/playstore)!
{% endtab %}

{% tab title="Beta" %}
1. Head to the [Google Play Store](https://www.lunarr.app/playstore)
2. Register for the test directly on the store listing
3. Download Lunarr via the Google Play Store
{% endtab %}

{% tab title="Edge" %}
1. Join the [Lunarr: Edge Testing](https://groups.google.com/g/lunasea-edge-test) Google Group
2. Head to the [Google Play Store](https://www.lunarr.app/playstore)
3. Register for the test directly on the store listing
4. Download Lunarr via the Google Play Store
{% endtab %}
{% endtabs %}
